# Medium

Given a square matrix[][]. The task is to rotate it by 90 degrees in clockwise direction without using any extra space.

Examples:

Input: mat[][] = [[1 2 3], [4 5 6], [7 8 9]]

Output:

7 4 1 

8 5 2

9 6 3

Input: mat[][] = [1 2], [3 4]

Output:

3 1 

4 2

Input: mat[][] = [[1]]

Output:

1


Constraints:

1 ≤ mat.size() ≤ 1000

1 <= mat[][] <= 100
